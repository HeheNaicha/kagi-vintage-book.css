# Vintage Book Theme - Kagi Search
Use Kagi Search feeling like you're reading a book. A world of knowledge at your fingertips.

### How to apply theme?
1. Go to Kagi
2. Toggle 'Light Mode' in the Control Center
3. Open 'Enable Custom CSS'
4. Apply kagi-vintage-book.css
5. Turn on Custom CSS in Control Center
